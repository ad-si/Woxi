# Comparison Operators

Relational comparisons: equality, inequality, ordering, and multi-way chains.


```scrut
$ wo 'x = 1; 0 <= x <= 2'
True
```

```scrut
$ wo 'x = 1; 0 < x < 2'
True
```

```scrut
$ wo 'x = 1; 0 > x < 2'
False
```

```scrut
$ wo 'x = 1; 0 < x > 2'
False
```
