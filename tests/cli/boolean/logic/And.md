# `And`

Logical AND operation.

```scrut
$ wo 'And[True, True]'
True
```

```scrut
$ wo 'And[True, False]'
False
```

```scrut
$ wo 'And[False, True]'
False
```

```scrut
$ wo 'And[False, False]'
False
```

```scrut
$ wo 'And[True, True, True]'
True
```

```scrut
$ wo 'And[True, False, True]'
False
```
