# `Xor`

Logical XOR operation.

```scrut
$ wo 'Xor[True, True]'
False
```

```scrut
$ wo 'Xor[True, False]'
True
```

```scrut
$ wo 'Xor[False, True]'
True
```

```scrut
$ wo 'Xor[False, False]'
False
```

```scrut
$ wo 'Xor[True, True, True]'
True
```

```scrut
$ wo 'Xor[True, True, False]'
False
```
