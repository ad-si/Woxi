# `GreaterEqual`

Check if values are greater than or equal to each other.

```scrut
$ wo 'GreaterEqual[2, 1]'
True
```

```scrut
$ wo 'GreaterEqual[1, 2]'
False
```

```scrut
$ wo 'GreaterEqual[1, 1]'
True
```

```scrut
$ wo 'GreaterEqual[1, 2, 3]'
False
```

```scrut
$ wo 'GreaterEqual[3, 2, 1]'
True
```


### `>=`

```scrut
$ wo '2 >= 1'
True
```

```scrut
$ wo '1 >= 2'
False
```
