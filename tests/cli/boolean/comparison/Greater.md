# `Greater`

Check if values are greater than each other.

```scrut
$ wo 'Greater[2, 1]'
True
```

```scrut
$ wo 'Greater[1, 2]'
False
```

```scrut
$ wo 'Greater[1, 1]'
False
```

```scrut
$ wo 'Greater[1, 2, 3]'
False
```

```scrut
$ wo 'Greater[3, 2, 1]'
True
```


### `>`

Check if values are greater than each other.

```scrut
$ wo '2 > 1'
True
```

```scrut
$ wo '1 > 2'
False
```
