# `Equal`

Compare values for equality.

```scrut
$ wo 'Equal[1, 1]'
True
```

```scrut
$ wo 'Equal[1, 2]'
False
```

```scrut
$ wo 'Equal[1, 1, 1]'
True
```


### `==`

Check if values are equal to each other.

```scrut
$ wo '2 == 2'
True
```

```scrut
$ wo 'x = 2; x == 2'
True
```

```scrut
$ wo '2 == 3'
False
```
