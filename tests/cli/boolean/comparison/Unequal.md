# `Unequal`

Compare values for inequality.

```scrut
$ wo 'Unequal[1, 1]'
False
```

```scrut
$ wo 'Unequal[1, 2]'
True
```

```scrut
$ wo 'Unequal[1, 1, 1]'
False
```


### `!=`

Check if values are not equal to each other.

```scrut
$ wo '2 != 2'
False
```

```scrut
$ wo 'x = 2; x != 2'
False
```

```scrut
$ wo '2 != 3'
True
```
