# `Gamma`

Gamma function. For positive integers, Gamma[n] = (n-1)!.

```scrut
$ wo 'Gamma[1]'
1
```

```scrut
$ wo 'Gamma[5]'
24
```

```scrut
$ wo 'Gamma[6]'
120
```

```scrut
$ wo 'Gamma[0]'
ComplexInfinity
```

```scrut
$ wo 'Gamma[-1]'
ComplexInfinity
```

```scrut
$ wo 'Gamma[x]'
Gamma[x]
```
