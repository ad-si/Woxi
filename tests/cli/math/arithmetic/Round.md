# `Round`

Rounds to the nearest integer.

```scrut
$ wo 'Round[3.5]'
4
```

```scrut
$ wo 'Round[3.4]'
3
```

```scrut
$ wo 'Round[-3.5]'
-4
```

```scrut
$ wo 'Round[-3.4]'
-3
```

```scrut
$ wo 'Round[0.5]'
0
```

```scrut
$ wo 'Round[-0.5]'
0
```

```scrut
$ wo 'Round[0]'
0
```

```scrut
$ wo 'Round[-0]'
0
```
