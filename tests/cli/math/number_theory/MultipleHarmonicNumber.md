# `MultipleHarmonicNumber`

The finite multiple harmonic sum over strictly decreasing indices
`n >= i1 > ... > ik >= 1` of `Product[1/ij^sj]`.

```scrut
$ wo 'MultipleHarmonicNumber[3, {2, 1}]'
5/12
```

A single weight reduces to the ordinary harmonic number.

```scrut
$ wo 'MultipleHarmonicNumber[6, {1}]'
49/20
```
