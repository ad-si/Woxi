# `Transliterate`

Transliterates text in other writing scripts to plain Latin/ASCII characters.

```scrut
$ wo 'Transliterate["Αλφαβητικός"]'
Alphabetikos
```

```scrut
$ wo 'Transliterate["алгоритм"]'
algoritm
```

```scrut
$ wo 'Transliterate["しんばし"]'
shinbashi
```

```scrut
$ wo 'Transliterate["안녕하세요"]'
annyeonghaseyo
```

```scrut
$ wo 'Transliterate["café"]'
cafe
```
