# Boolean Logic

Woxi implements the standard Wolfram Language boolean, comparison,
and predicate machinery. The documentation is split into three
topical pages:

- [Comparison Operators](boolean/comparison.md) — `Equal`, `Unequal`, `Greater`, `Less`, and their operator forms, plus multi-way comparisons.
- [Logical Operators and Conditionals](boolean/logic.md) — `And`, `Or`, `Not`, `Xor`, `If`, `Nand`, `Nor`, `Implies`, `SameQ`, `MatchQ`, …
- [Predicates](boolean/predicates.md) — all `*Q` predicates: `IntegerQ`, `EvenQ`, `OddQ`, `AllTrue`, `Positive`, `Negative`, `NumberQ`, `NumericQ`, `StringQ`, `ListQ`, `MatrixQ`, `PrimeQ`, `MemberQ`, `FreeQ`, …
