# `Mean`

Calculates the arithmetic mean (average) of a list.

```scrut
$ wo 'Mean[{1, 2, 3, 4, 5}]'
3
```

```scrut
$ wo 'Mean[{10, 20, 30}]'
20
```

```scrut
$ wo 'Mean[{1, 2, 3}]'
2
```

```scrut
$ wo 'Mean[{5}]'
5
```

```scrut
$ wo 'Mean[{-5, 5}]'
0
```

```scrut
$ wo 'Mean[{1.5, 2.5, 3.5}]'
2.5
```

```scrut
$ wo 'Mean[{0, 0, 0, 10}]'
5/2
```

```scrut
$ wo 'Mean[{-10, -5, 0, 5, 10}]'
0
```
